SpreadScripts

easyspreadread.py

- SUPPORTED FORMATS: .xls .xlt

- Usage: read large amounts of data from a spreadsheet and output the information in a separate file 
(output formats tested and accepted: .txt, .odt, .doc, .docx)

- Outputs cell data in the format of a list of lists where the innermost lists represent a row of 
information, for a 2x2 spreadsheet this would be in the format: 
[[column1/row1 value, column2/row1 value], [colum1/row2 value, column2 row2 value]]
